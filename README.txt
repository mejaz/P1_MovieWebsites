Movie Website - Movie Trailers Version 1.0 07/20/2015

Introduction:
-------------
This webpage displays different movie posters with the cast of the movie. Clicking on the posters plays the movie trailer.



Python Librarie(s) required:
----------------------------
webbrowser, os and re.



Database:
---------
None



Steps to run the file:
-------------------------------
1. The complete folder should have the following files :
	  i. fresh_tomatoes.py
	 ii. media.py
	iii. entertainment_center.py

2. Through command line, run the command : python entertainment.py
